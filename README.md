## 🇨🇱 Español

### Máquina vulnerable para práctica de hacking ético y CTF

**AndesBox** es una máquina Linux vulnerable creada para practicar ciberseguridad ofensiva en entornos controlados.
Diseñada para estudiantes, principiantes y personas que deseen mejorar habilidades reales de pentesting.

### Escenario

La empresa **Andes Logistics SPA** migró servicios internos sin aplicar controles de seguridad adecuados.
Su portal de empleados quedó expuesto con configuraciones débiles y vulnerabilidades explotables.

Tu objetivo será comprometer la máquina y capturar todas las flags.

**AndesBox** is a vulnerable Linux machine created for offensive cybersecurity practice in controlled environments.
